Создайте простое приложение для отслеживания расходов, чтобы управлять своими финансами.
Приложение должно позволять пользователям добавлять, удалять и просматривать свои расходы.
Приложение также должно предоставлять сводку расходов.

Требования

Приложение должно запускаться из командной строки и иметь следующие функции:

Пользователь может добавить расход с описанием и суммой.

Пользователь может обновить расход.

Пользователь может удалить расход.

Пользователь может просмотреть все расходы.

Пользователь может просмотреть сводку всех расходов.

Пользователь может просмотреть сводку расходов за конкретный месяц (текущего года).

Реализация

Вы можете реализовать приложение на любом языке программирования.
Вот несколько рекомендаций:

Используйте любой модуль/библиотеку для парсинга аргументов командной строки
(например: Python — argparse, Node.js — commander).

Используйте простой текстовый файл для хранения данных о расходах.
Это может быть JSON, CSV или любой другой формат.

Добавьте обработку ошибок:
— отрицательные суммы,
— несуществующие идентификаторы расходов,
— неверный ввод.

Разделяйте логику на функции — это упростит тестирование и поддержку.
